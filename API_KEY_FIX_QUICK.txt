# ⚡ QUICK FIX - API KEY NOT DETECTED

## 🚨 Problem
App shows: `⚠️  OPENAI_API_KEY not set. Speech recognition disabled.`

## ✅ Solution (Try in This Order)

### **FIX #1 - FASTEST (Try First)**

1. **Close ALL PowerShell windows completely**
   - Use Alt+F4 or X button
   - Close EVERY PowerShell/Terminal window

2. **Open a BRAND NEW PowerShell window**
   - Right-click desktop → PowerShell
   - OR Press Win+X → Terminal

3. **Copy and paste this exactly:**
```powershell
$env:OPENAI_API_KEY = "sk-proj-YOUR-ACTUAL-KEY-HERE"
cd "C:\Users\USR-LPTP-81\Desktop\zybertest-desktop"
npx tauri dev
```

Replace `sk-proj-YOUR-ACTUAL-KEY-HERE` with your real API key!

4. **Look for this message:**
```
✅ OpenAI Whisper API enabled
```

✅ **If you see this, it's FIXED!** Skip to testing below.

---

## ❌ Still not working?

### **FIX #2 - PERMANENT (Most Reliable)**

Set API key as **System Environment Variable** (works forever):

1. **Press:** `Win + X`
2. **Click:** `System`
3. **Click:** `Advanced system settings` (left side)
4. **Click:** `Environment Variables` (bottom right)
5. **Click:** `New...` (under User variables)
6. **Enter:**
   - Name: `OPENAI_API_KEY`
   - Value: `sk-proj-your-actual-key-here`
7. **Click:** OK → OK → OK
8. **Close all terminals and restart**
9. **Open new terminal and run:**
```powershell
cd "C:\Users\USR-LPTP-81\Desktop\zybertest-desktop"
npx tauri dev
```

✅ **Should now show:** `✅ OpenAI Whisper API enabled`
