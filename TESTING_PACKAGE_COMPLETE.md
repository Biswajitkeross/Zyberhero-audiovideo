## 🚨 Quick Troubleshooting

| Problem | Solution |
|---------|----------|
| No beep | Check Windows volume, enable system sounds |
| No terminal messages | Click "Start Monitoring" button |
| No audio detected | Check VoiceMeeter routing in Volume Mixer |
| API key error | Set: `$env:OPENAI_API_KEY="sk-..."` |
| Manual works, YouTube doesn't | VoiceMeeter routing issue - check mixer |

**Full troubleshooting in:** `QUICK_TEST_REFERENCE.md`
